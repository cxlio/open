# @cxl/spec-browser 
	
[![npm version](https://badge.fury.io/js/%40cxl%2Fspec-browser.svg)](https://badge.fury.io/js/%40cxl%2Fspec-browser)

Browser test runner and HTML reporter for spec tests. Renders results, error stacks, and supports click-to-run test re-execution.

-   Branch Version: [0.4.0](https://npmjs.com/package/@cxl/spec-browser/v/0.4.0)
