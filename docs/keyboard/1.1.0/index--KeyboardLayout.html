The KeyboardLayout interface describes how a keyboard layout should be structured internally.

Methods translateProperties modKeyshiftMapMethodstranslate(ev: Key): string

Describes a function that translates a Key object (representing a keyboard event) to a character string.

Parameters
  • ev: Key
PropertiesmodKey?: 'metaKey' | 'ctrlKey'

Indicates whether the 'meta' or 'ctrl' key is the primary modifier used for commands, differing based on the operating system (e.g., 'metaKey' for macOS).

shiftMap: Record<string, string | undefined>

A mapping of shifted keys to their non-shifted counterparts, used to accurately interpret which character a shifted key press represents.
